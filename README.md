# flutter_koin

![Flutter](https://img.shields.io/badge/Flutter-Dependency%20Injection-blue)  
**flutter_koin** — это DI (Dependency Injection) библиотека для Flutter, вдохновленная Koin.  
Она обеспечивает простое и гибкое управление зависимостями с поддержкой **кодогенерации**.

## 🚀 Основные возможности
- ✅ **Кодогенерация** для автоматической регистрации зависимостей.
- ✅ **Scoped зависимости** для гибкости управления объектами.
- ✅ **Простая настройка** без сложной конфигурации.
- ✅ **Минимальные зависимости** для максимальной производительности.
- ✅ **Полная совместимость** с iOS, Android, Web и Desktop.

---
